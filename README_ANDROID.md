# دليل تشغيل تطبيق KrootConnect على الأندرويد

للحصول على التطبيق بشكل نهائي على هاتفك، اتبع الخطوات التالية بدقة:

## الخطوة 1: تحميل المشروع
من القائمة الجانبية في AI Studio، اختر **Settings** ثم **Download ZIP** لتحميل المشروع كاملاً على جهاز الكمبيوتر الخاص بك.

## الخطوة 2: تجهيز واجهة المستخدم (React)
التطبيق يستخدم واجهة ويب متطورة، يجب بناؤها أولاً لتكون جاهزة للعمل بدون إنترنت:
1. افتح مجلد المشروع في أي محرر أكواد (مثل VS Code).
2. افتح "Terminal" ونفذ الأوامر التالية:
   ```bash
   npm install
   npm run build
   ```
3. بعد انتهاء الأمر، سيظهر مجلد جديد باسم `dist`.

## الخطوة 3: نقل الملفات إلى مشروع الأندرويد
1. اذهب إلى المجلد `android/app/src/main/`.
2. قم بإنشاء مجلد جديد باسم `assets` (إذا لم يكن موجوداً).
3. انسخ جميع محتويات مجلد `dist` الذي نتج من الخطوة السابقة وضعه داخل مجلد `assets`.
   * (يجب أن يكون ملف `index.html` موجوداً مباشرة داخل `assets`).

## الخطوة 4: فتح المشروع في Android Studio
1. افتح برنامج **Android Studio**.
2. اختر **Open** وقم بتحديد مجلد `android` الموجود داخل مشروعك.
3. انتظر حتى ينتهي البرنامج من إعداد الملفات (Gradle Sync).

## الخطوة 5: استخراج ملف التطبيق (APK)
1. من القائمة العلوية، اختر **Build**.
2. اختر **Build Bundle(s) / APK(s)** ثم اضغط على **Build APK(s)**.
3. بعد دقيقة، ستظهر رسالة في أسفل الشاشة تخبرك بانتهاء العملية، اضغط على كلمة **locate** للوصول لملف الـ APK.

## الخطوة 6: التثبيت على الهاتف
1. انقل ملف الـ APK الناتج إلى هاتفك.
2. قم بتثبيته (قد يطلب منك الهاتف السماح بالتثبيت من مصادر غير معروفة).
3. عند فتح التطبيق لأول مرة، سيطلب منك صلاحيات **SMS** و **البصمة**، وافق عليها ليعمل النظام بشكل صحيح.
